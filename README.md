# Berlín · 25–28 jul 2026

Guía-app de viaje a Berlín, instalable en el móvil (PWA) y con funcionamiento offline una vez cargada.

- Itinerario por días (Sáb 25 → Mar 28) en formato línea de tiempo.
- Restaurantes veganos o con opción vegana, comprobados como abiertos.
- Pestaña "Práctico": hotel, reservas (Reichstag, Berliner Unterwelten), transporte y recap.

Web: se sirve con GitHub Pages. En el móvil, "Añadir a pantalla de inicio" para usarla como app.
